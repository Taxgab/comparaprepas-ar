## 🛠️ Comandos Útiles

```bash
# Estado del sistema
openclaw status
openclaw gateway status

# Plugins
openclaw plugins list
openclaw plugins enable <nombre>
openclaw plugins disable <nombre>

# Agentes
openclaw agents list
openclaw agents list --bindings

# Security audit
openclaw security audit
openclaw security audit --deep

# Aprobaciones
openclaw approvals allowlist add "<comando>"
openclaw approvals get

# Doctor (diagnóstico)
openclaw doctor
openclaw doctor --fix
```

## 🔧 Troubleshooting Rápido

### Gateway no responde
```bash
openclaw gateway status
openclaw gateway restart
tail -50 /tmp/openclaw/openclaw-*.log
```

### Subagentes no funcionan
1. Verificar `openclaw.json` → `agents.list[].subagents.allowAgents`
2. Verificar plugin ACPX: `openclaw plugins list | grep acpx`
3. Usar `runtime: "subagent"` (no `"acp"` en Umbrel)

### Comandos piden aprobación
```bash
openclaw approvals allowlist add "<comando>"
openclaw approvals get
```

### Error de permisos
- Verificar `tools.elevated.enabled` en `openclaw.json`
- Verificar `tools.elevated.allowFrom` (Telegram ID)
